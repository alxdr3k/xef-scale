# ADR-0005: 광고 청정 정책 명문화

## Status

Accepted

## Date

2026-05-15

## Context

토스·뱅크샐러드 분석에서 가장 강하게 도출된 위험은 다음이다:

- **토스 P7** (광고 라벨링 정직): 시각 톤은 일반 카드와 동일하지만 "AD"/"서비스·광고" 표기. 단 9.8에서 외부 보도(미수거래·해외옵션 UX 논란)가 보여주듯 *친숙한 표현이 위험을 낮게 인식시키는 부작용* 발생 가능.
- **뱅샐 P10** (의미축의 광고 재활용 = dark pattern): "초강력 금융 혜택"의 금액에 긍정 컬러(민트)를 사용 → 광고와 절약 인사이트의 의미 구분 불가. 본 디스커버리에서 dark pattern으로 명시.

xef-scale은 현재 *광고가 없다*. 이 부재는 우연이 아니라 **신뢰 자산**이다. PRD(`PRD.md`)에도 광고는 명시되어 있지 않다(현 시점). 그러나 미래에 Phase C 수익 모델(`docs/design-phase-b.md`)에서 광고·제휴가 검토될 수 있다는 가능성이 디자인 결정에 그림자를 드리운다.

광고 부재는 다른 디자인 결정의 *자유도*를 만든다 — 특히 ADR-0006(의미축 분리)는 광고가 없기 때문에 가능하다(토스·뱅샐은 단일 강조색에 갇혀 있었다, `synthesis.md 2.4`).

## Decision

**xef-scale은 가계부 본면을 광고 청정으로 유지한다.** 명문화된 규칙:

1. **가계부 본면(홈/거래/검토함/카테고리)에 외부 금융상품 추천·광고를 노출하지 않는다.**
2. **광고를 도입하게 될 경우** (Phase C 수익 모델 등):
   - 광고는 별도 라벨("AD" 또는 "서비스·광고")로 명확히 표시.
   - 시맨틱 컬러(`action`, `positive`, `warning`)를 광고 영역에 사용하지 않는다.
   - 광고 영역은 *별도 탭* 또는 *별도 카드 톤*으로 분리.
   - 광고 도입 결정은 본 ADR을 supersede하는 새 ADR로 명시.
3. **추천·제휴 콘텐츠가 도입될 경우** (예: AI가 절약 팁 추천):
   - 콘텐츠가 *사용자 자신의 데이터에서 파생*된 것만 허용 (예: "이번 달 카페 지출이 늘었어요").
   - 외부 금융상품(카드·대출·보험·투자) 추천은 불가.
   - AI 출력은 ADR-0003의 AI 채널 격리(보라 + ✨)를 준수.

본 결정은 본 ADR을 채택하는 PR과 동일 PR에서 `PRD.md`의 "명시적 비기능 요구사항" 섹션에 한 줄로 반영한다.

## Consequences

**긍정**
- 신뢰 자산 보호 — 사용자는 가계부 화면에서 광고를 만나지 않는다.
- 디자인 자유도 확보 — ADR-0006의 의미축 분리가 가능해진다.
- AI 출력의 신뢰 격리(ADR-0003)와 톤이 일치 — "이 화면에는 외부 추천이 없다"는 일관된 기대.
- 가족 공유 환경(워크스페이스)에서 멤버 간 광고 노출 비대칭 문제 회피.

**부정**
- Phase C 수익 모델 옵션 축소.
- 미래에 광고 도입 시 본 ADR을 supersede하고 디자인 시스템 일부(특히 컬러)를 재조정해야 함.

**중립**
- 본 ADR은 *광고 영구 금지*가 아니다. 미래 결정을 위한 가드레일이며, supersede 가능.
- 사용자 자신의 데이터 기반 인사이트(VarianceCard 등)는 광고가 아니다.

**테스트/문서 영향**
- `PRD.md`에 광고 청정 한 줄 추가 (본 ADR 채택 PR과 동일 PR).
- view 코드 리뷰에서 외부 금융상품 노출이 들어가면 본 ADR 인용으로 거부.

## Alternatives considered

1. **광고를 처음부터 허용** — 거부. 토스·뱅샐 모두 광고로 디자인 자유도를 잃었다 (단일 강조색 강제). xef-scale은 광고가 없으므로 이 함정을 피한다.
2. **광고 가능성을 명시하지 않고 침묵** — 거부. 미래에 단일 PR로 광고가 슬그머니 들어올 위험. ADR로 가드레일 필요.
3. **모든 추천·제휴 금지** — 거부. 사용자 자신의 데이터 기반 인사이트(예: 절약 팁, 반복 결제 제안)는 *광고가 아니라 도구*이며 가계부의 가치 일부.
